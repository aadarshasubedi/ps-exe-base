ps-exe-base

Base library for Windows Form projects

View project on GitHub

Overview

This project is a base library for Windows Forms projects. It is used to start your exe and handles arguments or loading of the startup form. It includes classes to manage external processes, setting fonts on forms and all controls and handling updates to UI threads from background worker processes.

Start Up

The start up routine assumes there are two ways of invoking your executable - normally (shows a UI) and through the command line with arguments (no UI). Your project needs to use the Sub Main Startup option. In Sub Main you should invoke the PS.ExeBase.Starter.Start method passing an instance of a class defined in your project which implements the IManager class.
